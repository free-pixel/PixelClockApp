
这是一个番茄钟，可以给你的番茄钟设置不同的时间，task,break down,long break down.

每完成4个番茄钟，PixelClock会自动添加一个long break down time.

当计时器运行时，在mac的menu bar会同步显示进度条，红色进度条颜色会随时间逐渐充满整个圆圈。
